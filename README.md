# Spam Email Detector

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://spammemaildetector.streamlit.app)


### Project Overview
This project is a Machine Learning-based Spam Email Detector developed as part of my 4th Semester Macro Project. It allows users to input any email text and instantly classify it as Spam or Ham (Safe) using a highly accurate classification pipeline.

###  How It Works (The Algorithm)
The core of this project relies on a two-step Natural Language Processing (NLP) pipeline:

### 1. Feature Extraction: TF-IDF
Instead of just counting words, I used TF-IDF (Term Frequency-Inverse Document Frequency). 
TF: Measures how frequently a term occurs in a document.
IDF: Reduces the weight of terms that occur very frequently across all documents (like "the", "is", "of") and increases the weight of terms that are unique to spam (like "Winner", "Lottery", "Free").

### 2. Classification: Multinomial Naive Bayes (MNB)
I chose the Multinomial Naive Bayes algorithm for classification. It is a probabilistic learning method that is particularly effective for text-based datasets because:
* It is extremely fast to train and predict.
* It works well with the discrete features generated by TF-IDF.
* It follows Bayes' Theorem to calculate the probability of a message being spam based on the words it contains.
